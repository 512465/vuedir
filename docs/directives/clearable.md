# v-clearable

## 介绍

为输入框添加一个清除按钮，点击后可以快速清空内容。

## 基础用法

<ClearableDemo />

::: details 点击查看代码

<<< @/.vitepress/components/vClearable/ClearableDemo.vue

:::

## 注意事项

::: warning 注意

- 该指令只能用于 `input` 或 `textarea` 元素，不可用于组件。

:::

<script setup>
import ClearableDemo from '../.vitepress/components/vClearable/ClearableDemo.vue'
</script>
